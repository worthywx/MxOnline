
# MxOnline在线学习系统

### 搭建方法：  

<pre>
    1.下载最新版本
    2.Python2.7下安装使用到的包pip install -r requirements.txt
    3.修改settings中的DATABASES
    4.如果使用mysql，请在mysql中创建settings中配置的数据库
    5.python manage.py makemigrations
    6.python manage.py migrate
    7.python manage.py runserver 
    8.web页面中登录
</pre>
